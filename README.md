
# RustyShell

A command-line shell implementation written in Rust.

## Features

- Command parsing and execution
- Built-in shell commands
- Process management
- Error handling

## Getting Started

### Prerequisites

- Rust 1.56 or later
- Cargo

### Installation

```bash
git clone <repository-url>
cd RustyShell
cargo build --release
```

### Usage

```bash
cargo run
```
